K0NT4KT3

a lightweight, server-less, non-persistant contacts app to (temporarily) store contacts

Features:
- Add,
- Delete,
- Modify,
- Search,
- and sort Contacts

Contact Properties:
-metaData
    -timeStamp
    -UUID
-data
    -Name:
        -Surname and Name
    -Phone Number:
        -type
        -country Code
        -phone number
    -Address:
        -type
        -street name
        -house Number
        -zip
        -city
        -country
    -email
        -type
        -email address
    -bday
    -note
